ModelLoader

class ModelLoader(engine: Engine, context: Context)

Consumes a blob of glTF 2.0 content (either JSON or GLB) and produces a Model object, which is a bundle of Filament textures, vertex buffers, index buffers, etc. A Model is composed of 1 or more ModelInstance objects which contain entities and components.
