Creates a new anchor at the hit location.

See hitTests and com.google.ar.core.HitResult.getHitPose for details. Anchors incur ongoing processing overhead within ARCore. To release unneeded anchors use Anchor#detach().

This method is a convenience alias for {@code hitResult.getTrackable().createAnchor(hitResult.getHitPose())}
