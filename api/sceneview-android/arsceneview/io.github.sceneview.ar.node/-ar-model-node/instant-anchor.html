instantAnchor

var instantAnchor: Boolean = false
Content copied to clipboard

Anchor the node as soon as an AR position/rotation is found/available

If true, the node will be anchored in the real world at the first suitable place available. Depending on your need, you can change the placementMode to adjust between a quick (PlacementMode.INSTANT), more accurate (PlacementMode.DEPTH), only on planes/walls (PlacementMode.PLANE_HORIZONTAL, PlacementMode.PLANE_VERTICAL, PlacementMode.PLANE_HORIZONTAL_AND_VERTICAL) or auto refining accuracy (PlacementMode.BEST_AVAILABLE) placement.
