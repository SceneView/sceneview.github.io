How/where does the node is positioned in the real world

Depending on your need, you can change it to adjust between a quick (PlacementMode.INSTANT), more accurate (PlacementMode.DEPTH), only on planes/walls (PlacementMode.PLANE_HORIZONTAL, PlacementMode.PLANE_VERTICAL, PlacementMode.PLANE_HORIZONTAL_AND_VERTICAL) or auto refining accuracy (PlacementMode.BEST_AVAILABLE) placement. The hitTest, pose and anchor will be influenced by this choice.
