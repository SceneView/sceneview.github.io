ViewRenderable

@RequiresApi(api = Build.VERSION_CODES.N)
open class ViewRenderable : Renderable
Content copied to clipboard

Renders a 2D Android view in 3D space by attaching it to a Node with setModelInstance. By default, the size of the view is 1 meter in the SceneView per 250dp in the layout. Use a ViewSizer to control how the size of the view in the is calculated. 


future = ViewRenderable.builder().setView(context, R.layout.view).build();
viewRenderable = future.thenAccept(...);
