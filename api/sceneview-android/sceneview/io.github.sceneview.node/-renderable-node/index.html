RenderableNode

open class RenderableNode(val renderables: List<Renderable> = listOf()) : Node
Content copied to clipboard

A Node represents a transformation within the scene graph's hierarchy.

This node contains a renderable model for the rendering engine to render.

Each node can have an arbitrary number of child nodes and one parent. The parent may be another node, or the SceneView .
