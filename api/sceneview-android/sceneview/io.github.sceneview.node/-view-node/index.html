ViewNode

open class ViewNode(val engine: Engine, val modelLoader: ModelLoader, val viewAttachmentManager: ViewAttachmentManager, parent: Node? = null) : Node

A Node represents a transformation within the scene graph's hierarchy.

This node contains a View for the rendering engine to render.

Each node can have an arbitrary number of child nodes and one parent. The parent may be another node, or the scene.
