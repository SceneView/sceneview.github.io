fun RenderableManager.Builder.geometry(geometry: Geometry): RenderableManager.Builder
Content copied to clipboard

Specifies the geometry data for a primitive.

Filament primitives must have an associated VertexBuffer and IndexBuffer. Typically, each primitive is specified with a pair of daisy-chained calls: geometry and RenderableManager.Builder.material.
